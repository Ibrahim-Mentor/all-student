Shopify Development

Build and customize e-commerce stores using the powerful Shopify platform.

About this course

Take your first steps into e-commerce development. This course covers everything from setting up a new Shopify store to understanding its theme architecture. You'll learn how to manage products, collections, and settings from the admin dashboard and make basic code edits to themes.

What you'll learn

  • Set up and configure a new Shopify store from scratch.
  • Understand the Shopify dashboard, products, and collections.
  • Learn the basics of the Liquid templating language.
  • Make basic customizations to existing Shopify themes.
